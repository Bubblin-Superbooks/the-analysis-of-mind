The changes in appearances of an object which are due to changes in the intervening medium will not affect, or will affect only very slightly, the appearances from places close to the object. If the appearances from sufficiently neighbouring places are either wholly un changed, or changed to a diminishing extent which has zero for its limit, it is usually found that the changes can be accounted for by changes in objects which are between the object in question and the places from which its appearance has changed appreciably.
